Print Formats Picking CRM

Este módulo permite la impresión del informe CMR a partir de un albarán dado.

Tabla de contenidos

  • Utilización
  • Créditos
    • Autor

Utilización

  • Rellenamos el campo "Código CMR".
  • Una vez estemos en el albarán haremos click en el desplegable "Imprimir" y a continuación seleccionaremos "Informe CMR".
